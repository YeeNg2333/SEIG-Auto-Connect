## 特色
- [x] 支持一键登录网页端校园网账号
- [x] 支持自动获取登录参数
- [x] 自动登录功能，在启动软件时自动登录保存的账号
- [x] 自动识别验证码，如果失败5次，需要手动输入验证码
- [x] 看门狗，默认每600秒检测一次网络状态，若网络不通，自动重连
- [x] 对保存的密码低级加密
